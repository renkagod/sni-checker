# SNI List

Коллекция SNI доменов, полученная путем автоматизированного сканирования и фильтрации.

## Описание
В данном списке собраны домены, прошедшие проверку на доступность и наличие TLS-сертификатов. Сбор данных осуществлялся с помощью инструментов `subfinder` (сбор поддоменов через различные источники) и `httpx` (фильтрация по статус-кодам и проверка TLS).

## Состав
- `sni.txt`: Список доменов, отфильтрованных по статус-кодам (200, 403, 404) и имеющих активный TLS.
