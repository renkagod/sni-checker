# SNI List

Коллекция SNI доменов, полученная путем автоматизированного сканирования и фильтрации.

## Описание
В данном списке собраны домены, прошедшие проверку на доступность и наличие TLS-сертификатов. Сбор данных осуществлялся с помощью инструментов `subfinder` и `httpx`.

## Состав
- `sni.txt`: Список доменов с активным TLS.
